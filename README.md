# Panto Landing Page

![Panto Landing Page Thumbnail](./public/thumbnail.png)

A modern, responsive landing page for Panto, a furniture e-commerce platform built with React and Vite. This project showcases a sleek design with interactive components, shopping cart functionality, and a user-friendly interface.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices using Tailwind CSS.
- **Interactive Components**: Includes hero sections, testimonials, product showcases, and more.
- **Shopping Cart**: Integrated cart context for managing products.
- **Theme Support**: Dark and light theme toggle.
- **Routing**: Client-side routing with React Router.
- **Image Optimization**: Utility functions for handling image URLs.
- **User Feedback**: SweetAlert2 for alerts and confirmations.

## Pages

- **Home**: Hero section, experiences, materials, testimonials, why choose us.
- **About**: Information about the company.
- **Contact**: Contact form and details.
- **Shop**: Product listings and individual product cards.

## Technologies Used

- **React**: Frontend library for building user interfaces.
- **Vite**: Fast build tool and development server.
- **Tailwind CSS**: Utility-first CSS framework.
- **React Router**: Declarative routing for React.
- **Swiper**: Modern touch slider.
- **React Icons**: Popular icons library.
- **SweetAlert2**: Beautiful, responsive alerts.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/panto-landing-page.git
   cd panto-landing-page
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Start the development server:
   ```bash
   pnpm dev
   ```

4. Open your browser and navigate to `http://localhost:5173`.

## Usage

- Navigate through the pages using the navbar.
- Add products to the cart from the shop page.
- Toggle between light and dark themes.
- View testimonials and product details.

## Build

To build the project for production:

```bash
pnpm build
```

Preview the production build:

```bash
pnpm preview
```
